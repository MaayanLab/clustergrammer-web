clustergrammer

Clustergrammer is a visualization tool that enables users to easily generate highly interactive and shareable clustergram/heatmap visualizations from a matrix of their own data. Press play or explore the example below to see the interactive features.

See example in full page

Your tab-separated matrix file should be in one of the supported matrix formats and have a .txt or .tsv file extension. See example simple matrix format below: 


       Col-A   Col-B   Col-C
Row-A   0.0    -0.1     1.0
Row-B   3.0     0.0     8.0
Row-C   0.2     0.1     2.5

Once uploaded you will obtain a permanent and shareable link to your visualization.

Please see Clustergrammer's Documentation or Getting Started for more information.

Case Studies

CCLE Explorer

Clustergrammer was used to re-analyze the Cancer Cell Line Encyclopedia's (CCLE) gene expression data and enable tissue-specific exploration with the CCLE Explorer. Also see the corresponding Clustergrammer CCLE Jupyter Notebook.

Lung Cancer PTM/Expression Data

The Clustergrammer-Widget was used to analyze lung cancer regulation at the post-translational modification (PTM) and gene expression levels (see Notebook). PTM data was obtained from our collaborators at Cell Signaling Technology.

Single-Cell CyTOF Data

Single-Cell CyTOF was used to investigate the phosphorylation response of peripheral blood mononuclear cells (PBMCs) to PMA (phorbol 12-myristate 13-acetate) (see Notebook ). Data obtained from collaborators at the Icahn School of Medicine Human Immune Monitoring Core.

Kinase Substrate Similarity Network

The Kinase Substrate Similarity Network uses Clustergrammer to visualize a large network of kinases based on shared substrates.

MNIST Handwritten Digits

The Clustergrammer-Widget was used to visualize the MNIST handwritten digit dataset (see Notebook). K-means downsampling was used to visualize the entire MNIST dataset as a single heatmap.

Iris Dataset

The Clustergrammer-Widget was used to visualize popular the Iris flower dataset. See the Clustergrammer Iris Jupyter Notebook.

Using Clustergrammer

Jupyter Notebook Widget

The Clustergrammer-Widget enables users to easily generate interactive visualizations within shareable Jupter notebooks (see example). Jupyter notebooks are ideal for generating reproducible workflows and can be shared using nbviewer

Biology Specific Features

Clustergrammer is also available as an IPython interactive widget. Clustergrammer-Widget enables users to produce interactive heatmaps as part of their workflow and share their interactive visualizations using Jupter's nbviewer

Open Source Code for Developers

Clustergrammer is a free and open source project available through GitHub. The front-end JavaScript library, back-end Python library, and documentation can be found via the link above.

Web App Integration Examples

Enrichr

Clustergrammer visualizes enrichment analysis results from the Ma'ayan lab web-tool Enrichr and displays the overlap of a user's input gene list and the gene lists of enriched terms.

GEN3VA

Clustergrammer visualizes collections of gene expression signatures extracted from the Gene Expression Omnibus (GEO) using the Ma'ayan lab web-tool GEN3VA.

L1000CDS2

Clustergrammer visualizes gene-expression-signature search results from the Ma'ayan lab web tool L1000CDS2, which allows users to find perturbations from the L1000 project whose signatures are similar or antisimilar to their input signature.

Clustergrammer is being developed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai for the BD2K-LINCS DCIC and the KMC-IDG
and is an open source project available on GitHub: Clustergrammer, Clustergrammer-web, and Clustergrammer-widget.
