Summary

Clustergrammer is a web-based visualization tool that allows users and web-based applications to easily visualize a matrix of data as an interactive and shareable clustergram-heatmap. Clustergrammer simplifies the task of hierarchical clustering - which is commonly used in biology to study high-throughput data - and has no software requirements other than a modern browser (e.g. Chrome). Clustergrammer allows users to intuitively interact with their visualization - by zooming, reordering and filtering - and easily share their interactive visualizations on the web with collaborators. Clustergrammer is built using the JavaScript visualization library Data Driven Documents (D3.js) and is capable of visualizing large datasets (on the order of 100,000 data points), which cannot be effectively visualized using static images. Clustergrammer provides dynamic visualizations for the Ma'ayan Lab web-tools: Enrichr, Gen3va, L1000CDS2, and Harmonizome. Developers can also use the Clustergrammer API to produce dynamic visualizations for their own projects or view the open-source code on Github.

Tutorial

Upload Matrix

Making an interactive and shareable clustergram visualization using Clustergrammer simply requires uploading a matrix of data at the homepage. A user's uploaded matrix file should have the format shown in Fig. 1.

and have a .txt or .tsv file extension. Once uploaded the user will be redirected to a visualization page where they can interact with their matrix visualization. The visualization page URL can also be used to return to the visualization at a later time or to share it with others.

Note that the user's data does not need to be clustered before uploading. Also, if a user is uploading an asymmetrical matrix, e.g. a matrix with more columns than rows, it is recommended that the user transpose the matrix if necessary in order to have more rows than columns. This will provide the best results in terms of zooming and filterings.

Data Processing

Clustergrammer aims to extend the capabilities of the clustergram visualization by providing users with a multitude of interchangable views of their data - e.g. views with different orderings and filters. Clustergrammer creates these 'views' of a users data in the following manner. First, Clustergrammer calculates a 'Cluster' ordering of rows and columns by performing hierarchical clustering of the rows and columns of a user's data (clustering is done using cosine distance and average linkage using the Python package SciPy). Groups of similar rows and columns are calculated at different distance thresholds to produce an interactive dendrogram-like colorbar. Second, Clustergrammer calculates a 'Rank' ordering of rows and columns by ranking the rows and columns based on the sum of their values.

Third, the previous ordering calculations are repeated on successively filtered versions of the matrix. Clustergrammer start by finding the row with the greatest absolute value sum and this row-maximum-value is used to filter out rows from the matrix. Clustergrammer filters out rows that have an absolute value sum of less than some percentage of the row-maximum-value - where the percentage threshold varies from 0% to 90% in 10% increments. Finally, a user can dynamically create new views of their data by reordering the matrix based on the values in any row or column by double-clicking a row or column label. Collectively, these different orderings and filterings give the user a large set of interchangable views of their data.

Visualization

The Clustergrammer visualization has many interactive features including:

  • Zooming and Panning
  • Reordering
  • Row Search
  • Interactive Row/Column Dendrogram-like Colorbar
  • Row Filtering
