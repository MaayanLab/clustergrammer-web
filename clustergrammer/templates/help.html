Summary

Clustergrammer is a web-based visualization tool that allows users and web-based appliactions to easily visualize a matrix of data as an interactive and shareable clustergram-heatmap. Clustergrammer simplifies the task of hierarchical clustering - which is commonly used in biology to study high-throughput data - and has no software requirements other than a modern browser (e.g. Chrome). Clustergrammer allows users to intuitively interact with their visualization - by zooming, reordering and filtering - and easily share their interactive visualizations on the web with collaborators. Clustergrammer is built using the JavaScript visualization library Data Driven Documents (D3.js) and is capable of visualizing large datasets (on the order of 100,000 data points), which cannot be effectively visualized using static images. Clustergrammer is being used to provide dynamic visualizations for the Ma'ayan Lab web-tools: Enrichr, Gen3va, and Harmonizome. Developers can also use the Clustergrammer API to produce dynamic visualizations to embed in their own projects or view the open-source code on Github.

Tutorial

Upload Matrix

Making an interactive and shareable clustergram using Clustergrammer only requires uploading a matrix of data. A user's uploaded matrix file should have the following format: 

 
       Col-A   Col-B   Col-C
Row-A   0.0    -0.1     1.0
Row-B   3.0     0.0     8.0
Row-C   0.2     0.1     2.5

and have a .txt or .tsv file extension. Once uploaded the user will be redirected to a visualization page where they can interact with their matrix visualization. The visualization page URL can also be used to return to the visualization at a later time or to share it with others.

Data Processing

Clustergrammer

Clustergrammer processes a user's uploaded data on the server side using SciPy. Once uploaded a users matrix of data is clustered (using cosine distance and average linkage), and the rows and columns are ranked based on their sum. This process is repeated 10 times using different thresholds for row-data filtering: rows are filtered out if the absolute value of their sum falls below a given threshold. This threshold is set in increments of 0.1 between 0 and 1 times the maximum absolute value sum of the rows. Clustergrammer also calculates row and column group information at different distance thresholds in order to produce an interactive dendrogram-like colorbar.
